# Quad

## Syntax

```typescript
type Quad = [
	[ number,number ], // Point one
	[ number,number ], // Point two
	[ number,number ], // Point three
	[ number,number ], // Point four
];
```

**Remark**

The four points should be arranged in a clockwise or counterclockwise  order.
